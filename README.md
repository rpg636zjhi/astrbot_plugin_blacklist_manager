# 黑名单管理(blacklist_manager)
> 该插件提供了用户和群组的黑名单管理功能，可以拦截黑名单用户或群组的消息，确保机器人运行环境的安全性和可控性。
# 功能特点
- 自动退群：当Bot检测到自己在群黑名单中时自动退群

- 用户警告：当黑名单用户在群内发言时，会@该用户并发送警告消息

- 私聊拦截：黑名单用户的私聊消息会被拦截

# 可用指令
### 用户黑名单管理
- /黑名单 add QQ号 - 添加用户到黑名单

- /黑名单 remove QQ号 - 从黑名单移除用户

- /黑名单 list - 查看用户黑名单列表

### 群黑名单管理
- /群黑名单 add 群号 - 添加群到黑名单

- /群黑名单 remove 群号 - 从群黑名单移除群

- /群黑名单 list - 查看群黑名单列表

### 配置说明
插件支持以下配置选项（通过管理面板配置）：

auto_leave_group: 是否自动退出黑名单群

warn_message: 黑名单用户警告消息内容

## 注意事项
该插件专门为QQ平台设计，使用了aiocqhttp平台适配器

数据存储在 data/blacklist_data/ 目录下

使用时请确保Bot运行在QQ平台，并且具有管理员权限的用户才能管理黑名单。
